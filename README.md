# study-rust

## install

```shell
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## how to use cargo
### create project

```shell
cargo new [project name]
```

### build(compile) & run

```shell
cargo run
```

### build(compile)

```shell
cargo build [--release]
```
